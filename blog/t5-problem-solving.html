Can you Tell me What .map() does?

From what I found, It is used to convert one array into another. it takes the values in an existing array and uses a small piece of code called a function to manipulate those values in some way, and create a new array with the new values. The .map() method could be used by a function to take some numbers out of one array, multiply all those numbers by two, and then put the results inside a new array. This is a very basic example.

How about .filter()?

is similar to the .map() method. The difference is that when the function looks through the existing array, it creates the new array only with values that pass a test. Some examples: it may look through some names and return those that start with a particular letter; or it may only return values that are greater than 100. It will then put the values it finds in a new array - it doesn't actually change the values as .map() does.

And .reduce()?

takes the values in an array and reduces them to a single value. It can do this with numbers by adding them all together (or any calculation that uses two or more values). It can take an array with a bunch of smaller arrays inside it, take the values out of those and turn all of them into one big array. It can also take words inside an array and stick them all together in a big sentence. The beauty of .reduce() is that it simplifies a lot of the sort of basic coding that would spit out a single value anyway.
